## zzjs-cli  脚手架工具

`zz.js` 是一个开源项目，使用koa/react/babel/webpack搭建，可以实现服务端渲染的场景，同时也可以进行同构开发。

`zzjs-cli` 是一个命令行工具， 可以方便的帮助你创建基础项目和项目中的page 页面，帮你提高效率。

## 使用说明

`npm i zzj-cli -g `  全局安装

`zzj -h` 查看命令帮助

`zzjs -i` 根据提示可创建项目和项目内的页面,创建页面需要切换到项目的根目录
